# Archive Legacy

Archivio storico non canonico.

Non usare questi documenti come fonte di verita tecnica corrente. Sono report,
audit, roadmap superate, snapshot e note operative conservate solo per
ricostruire il contesto storico delle decisioni passate.

Questi documenti non devono guidare nuove decisioni tecniche. Se un contenuto in
questo archivio sembra contraddire un documento canonico, prevale sempre il
documento canonico.

Fonti canoniche correnti:

- `docs/architetture/**`
- `docs/seo-navigation/**`
- `docs/taxonomy.md`
- `docs/design/HOME_DESIGN_SYSTEM.md`
- pochi runbook operativi attivi, come `docs/setup/SUPER_ADMIN_BOOTSTRAP.md`

Per nuove modifiche architetturali, leggere prima le fonti canoniche e trattare
questo archivio solo come memoria storica.
